## Why Heart Disease?

Honestly, I wanted something that actually matters. Predicting 
cricket scores is fun, but this felt more real. 1025 patients. 
14 medical attributes. One question: does this person have 
heart disease or not? That felt worth building.

## What This Project Does

Takes patient data — age, cholesterol, blood pressure, chest 
pain type, and more — and predicts whether they have heart 
disease. Binary classification: 0 means no disease, 1 means 
disease present.

## Dataset
- UCI Heart Disease Dataset (via Kaggle)
- 1025 patients, 14 features
- Almost perfectly balanced — 51.3% disease, 48.7% no disease
